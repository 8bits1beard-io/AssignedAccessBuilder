Deploy to Intune via OMA-URI

Steps to Create Custom CSP Policy

1. Navigate to Intune

  • Go to Microsoft Intune admin center
  • Select Devices â†’ Configuration profiles

2. Create Profile

  • Click + Create profile
  • Platform: Windows 10 and later
  • Profile type: Templates â†’ Custom
  • Click Create

3. Configure OMA-URI Setting

  • Name the profile (e.g., "Kiosk Configuration")
  • Click Add under OMA-URI Settings
  • Fill in the following:
FieldValue
NameAssignedAccess Configuration
OMA-URI./Vendor/MSFT/AssignedAccess/Configuration
Data typeString
Value(Paste your XML here)

4. Assign & Deploy

  • Assign to a device group
  • Device will reboot to apply kiosk mode

Important Notes

  • Device must be Azure AD joined or Hybrid Azure AD joined
  • User account must exist on the device before policy applies
  • Test on a single device before broad deployment
